FeatureInformation¶

class s100py.s111.FeatureInformation(recursively_create_children=False, **kywrds)¶

Bases: s100py.s1xx.S1XX_Attributes_base, abc.ABC

S111 10.2.2 and Table 10.3 of v1.0.1 and S100 Table 10c-8 v4.0.0 This is used to describe the SurfaceCurrent within the GroupF feature listing. The features described under GroupF have a matching named entry parallel to GroupF (top level). The actual data (surfaceCurrentSpeed etc) is stored in the top level element while basic metadata is stored in this element.
